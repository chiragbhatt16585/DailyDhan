# Logo Files

This folder contains logo images for affiliate platforms:

- `amazon-logo.webp` ✅ (WebP format)
- `flipkart-logo.png` ✅ (PNG format)
- `myntra-logo.jpg` ✅ (JPEG format)

## Logo Details

All logos are configured and will be displayed in the Affiliate Accounts screen with the same styling as Amazon (white background, 40x40px display size).

The logos are automatically loaded based on the platform ID in the affiliate accounts configuration.
